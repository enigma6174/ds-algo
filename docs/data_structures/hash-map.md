Represent data as key value pairs. To access the value we have to index with the key.

In JavaScript they are represented with Object Literals and in Python they are represented with Dict

| Operation | Time |
| :-------: | :--: |
| Insertion | O(1) |
|  Lookup   | O(1) |
|  Search   | O(1) |
| Deletion  | O(1) |

**Note**
Due to collisions some operations like may take up O(n) time

### Positives

- Fast Lookup (good collision resolution needed)
- Fast inserts
- Flexible keys

### Negatives

- Unordered
- Slow
