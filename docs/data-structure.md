# Introduction To Data Structures

It is a container to store data. This stored data is used by **algorithms** later on to execute a program. Data structures simply help us store and represent the data in an efficient manner. Array, string, stack, queue, linked list, tree etc. are some of the most popular data structures available.  

> A data structure is a container to store and represent data efficiently

## Operations On Data Structures

### Insertion

Inserting a new item into the data structure

### Deletetion

Removing an item from the data structure

### Searching

Finding an item in the data structure

### Traversal

Access each item of the data structure once 

### Sorting

Representing the data in an ordered way - increasing order or decreasing order

### Accessing

How to access an individual item of the data structure
